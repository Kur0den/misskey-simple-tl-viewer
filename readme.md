## 使い方

### windows のひと

1. Releases から exe ファイルをダウンロード
2. 実行

### 自分でいろいろしたいひと

1. clone
2. 環境作成
3. run main.py

    ホームタイムライン

- l

    ローカルタイムライン

- s

    ソーシャルタイムライン

- g

    グローバルタイムライン
